## $\color{#c52233}{\text{Project Idea: Stylized Blog Article}}$

With the help of prompts to Google Gemini, I have a list of five CSS project ideas to creater as I learn basic CSS along with the CSS Foundations>Intro to CSS section of The Odin Project Foundations. This is project idea #1 - a blog article.

I decided to clone the blog page for Hyde (a Jekyll theme). I cloned the Hyde blog featured in this Medium article: [link](https://daveyem.medium.com/practising-css-cloning-a-web-site-64b10870d151).

Using a table for the basic structure, I created a version of the theme's blog that uses CSS to look as close as possible.

- **HTML Structure**: One `<h1>` for the title, an image, a few `<h2>` subtitles, paragraphs `<p>`, and an unordered list `<ul>` for ingredients or bullet points. Link Credits per Google Gemini: [[1](https://www.skillshare.com/en/classes/code-your-own-portfolio-html-and-css-basics/1042721241), [2](https://www.reddit.com/r/learnprogramming/comments/106i6ry/i_am_felling_completely_clueless_with_css_odin/)]

- **CSS Goals**:
    - Change the background color of the entire page using the `body` selector.
    - Center the main title and give it a unique color.
    - Style the text paragraphs with a clean, readable font size.
    - Change the color of the bullet points or list items to make them stand out. Link Credits per Google Gemini:[[1](https://blog.devgenius.io/understand-the-basics-of-a-website-frontend-html-css-js-a2bc011081e5), [2](https://dev.to/fullstackjo/css-tutorial-series-css-challenge-1-4coc), [3](https://www.youtube.com/watch?v=eVVi2-td9B8&t=278), [4](https://www.youtube.com/watch?v=6zVJWDJIsgY&t=419)]
