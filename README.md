🎨 # Project Idea 1: Stylized Wiki Recipe or Blog Article

Instead of standard plain black-and-white HTML text, transform a basic text document into a beautiful, readable editorial article.

- **HTML Structure**: One `<h1>` for the title, an image, a few `<h2>` subtitles, paragraphs `<p>`, and an unordered list `<ul>` for ingredients or bullet points. [[1](https://www.skillshare.com/en/classes/code-your-own-portfolio-html-and-css-basics/1042721241), [2](https://www.reddit.com/r/learnprogramming/comments/106i6ry/i_am_felling_completely_clueless_with_css_odin/)]

- **CSS Goals**:
    - Change the background color of the entire page using the `body` selector.
    - Center the main title and give it a unique color.
    - Style the text paragraphs with a clean, readable font size.
    - Change the color of the bullet points or list items to make them stand out.
